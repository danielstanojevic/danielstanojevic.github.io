CSS Spacing

May 24th, 2015

What is the difference between Margin, Border, and Padding? Every html element has 3 different buffers of space around it. Margin is the spacing around an element outside of the border. Border is there to define the border attributes, such as style, and thickness. Padding is the spacing around an element inside of the border, but just outside of the element. A great image can be seen in Chrome DevTools. This is also a great tool to make sure you're selecting the right attribute to modify.

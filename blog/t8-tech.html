Recursion is a very powerful function in programming. To use it, you need to recognize when you can create a base case where you know what the result of calling the function will be. You also need to have some way of moving toward that base case. Then, it's a matter of calling the function on itself. It can be quite confusing, sort of like the movie Inception, but with practice, you can get the hang of it and it will save you time in the long run if you can recognize when to use it.

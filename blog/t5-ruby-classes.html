Ruby classes

June 21st, 2015

Classes and objects are quite powerful data structures used to model real world objects in our computer programs. An object is a particular instance of a class. For example, Lassie is an object in the class Dog. Fido would be another object in the class Dog. Lassie's bark method may result in "Woof!" while Fido's bark might result in "Yelp!".

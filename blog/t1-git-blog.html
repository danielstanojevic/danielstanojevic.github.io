Reflections on version control

git and GitHub

May 24th, 2015

What are the benefits of version control? There are many benefits of version control. Version control can be used to create a backup. It's helpful with collaboritive work flow. It can make debugging a program easier by keeping track of changes in logical chunks. There are many other benefits as well as the ones mentioned.

How does git help you keep track of changes? git helps you keep track of changes by allowing you to create a save point or multiple save points if you like.

Why use GitHub to store your code? It's helpful if you lose your computer, you still have access to your code online. Plus, if you intend to share your code, GitHub is a great place to to do so.

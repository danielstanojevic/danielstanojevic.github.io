Arrays and Hashes

June 4th, 2015

Describe and compare arrays and hashes, including Ruby code. Arrays and Hashes are both data structures. They are indexed differently though where an array is indexed by an integer, hashes are indexed by objects or symbols. A dictionary provides a great example of when it would be better to use a hash instead of an array. It would be difficult to access a particular word in an array because we'd have to know the proper index number to access it. However, with a hash, the word is the index, which provides very easy access to the word definition or translation that we're looking for in our dictionary, for example. An array can store any object To create a new array versus a new hash:
